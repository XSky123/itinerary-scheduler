# 📖 行程安排工具：看图就会的超简单教程

> 不怕，点错了就按 `Ctrl + Z`，或点“↩”。大部分操作都可以撤销。

## 第 1 步：打开网页

点这里：[打开行程安排工具](https://xsky123.github.io/itinerary-scheduler/)

第一次打开时，屏幕上已经放好了一张根室玩具时刻表： **5 班火车** + **6 班公交** 。已经选好的方案是“其中 1 班火车 + 其中 1 班公交”。

**注意：** 示例时间只用来教你操作，不是实时交通时刻表。真的要出发时，请按官方信息修改。

![电脑上的默认根室时刻表](images/01-sample-overview.png)

## 第 2 步：认识三个大盒子

电脑上从左到右看：

1. **左边“班次库”** ：放整张火车、公交或飞机时刻表。
2. **中间“时刻表”** ：看所有候选班次，再选出最终方案。
3. **右边“行程预览”** ：每一段的出发和到达时间放在同一行。

手机上三个盒子会从上到下排队，往下滑就能看到。

![手机上的上下排列](images/02-phone-layout.png)

## 第 3 步：一次导入整张时刻表

1. 点左上角的 **“批量导入”** 。
2. 选“火车”或“巴士”，再选要放入的交通行。
3. 每行粘贴一班，写成：

```text
09:00-11:30 JR 花咲线
11:15-13:45 JR 花咲线
13:25-15:55 JR 花咲线
```

4. 点 **“导入整张时刻表”** 。

只想加一班时，点 **“＋ 单班”** 。

## 第 4 步：从 N 班里选 1 班

1. 在中间的候选班次块上点 **鼠标右键** 。
2. 点你想加入的方案名字。
3. 同一交通行再选另一班时，新班次会自动换掉旧班次。

例如“火车 5 选 1”，一个方案里只会保留 1 班火车。公交行也是一样。

## 第 5 步：改班次或拖动时间

1. 用鼠标 **点一下** 班次块，左边会打开编辑。
2. 抓住块的 **中间** ：整块向左或向右移。
3. 抓住块的 **左边或右边** ：把时间变长或变短。

每次会自动对齐到 5 分钟，不用担心拖歪。

## 第 6 步：加一个“吃饭”或“休息”

1. 找到“最终方案”里两个班次中间的空白。
2. 在空白处点 **鼠标右键** 。
3. 输入“吃饭”、“买票”或“休息”。

事项不能压住火车或其他事项。拖到边界时，它会自己停下来。

## 第 7 步：遇到重叠怎么办

- **上面的候选班次碰在一起** ：可以，它们会自动上下分开。
- **不同交通行的已选班次重叠** ：会变红，提醒这个接续不可行。
- **事项碰到班次** ：不让它们重叠，会停在空白处。

## 第 8 步：一键清空，以及后悔药

1. 点时刻表上方的红色 **“一键清空”** 。
2. 班次、交通行、事项和方案会一起清空。
3. 后悔了？立刻点它左边的 **“↩”** ，或按 `Ctrl + Z`。

## 第 9 步：做成好看的行程单

1. 看右下角。
2. 点绿色的 **“导出精美行程单”** 。
3. 导出文件中，每段交通的起止时间放在同一行。
4. 想要 PDF？打开 HTML 后按 `Ctrl + P`，选“另存为 PDF”。

![起止时间放在同一行的导出行程单](images/03-export-report.png)

## 最后只记住三句话

1. **左边导入整张时刻表。**
2. **中间每行 N 选 1。**
3. **点错就按 `Ctrl + Z`。**

发现怪问题时，请在 [GitHub Issues](https://github.com/XSky123/itinerary-scheduler/issues) 告诉我们。
